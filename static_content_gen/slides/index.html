Generación de contenido estático

¿Que es el contenido estático?

Es aquel contenido que no cambia o que no depende de otras cosas.

Por el contrario, el contenido dinámico que depende de variables como el tiempo o input provisto por usuario, etc

Ejemplos

Estático

  • Imágenes
  • Codigo html, javascript, css
  • Archivos

Dinámico

  • Likes, comentarios
  • Respuestas de formularios
  • Chats
